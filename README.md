# ShiftBoard

This is a template for a new Vite project with React, TypeScript, and shadcn/ui.

---

## Setup Steps

1. Create a Supabase project.
2. Enable Google OAuth provider in Supabase Auth settings.
3. Run database migrations to create `users` and `shifts` tables with RLS policies.
4. Seed the admin user email.
5. Create a Resend account and verify sender domain/email.
6. Deploy the Supabase Edge Function for email sending.
7. Scaffold the Vite + React project with TanStack Router.
8. Wire up Supabase client, auth, and shift CRUD.
9. Build the UI.
10. Deploy to Vercel.
